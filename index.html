GITHUB

Primeros pasos

Estoy iniciando un desarrollo de una app y ¿? eureka

porque no mejor la conecto (subo) a GITHUB

  • Ver status
    • git status
  • Ver si ya estamos conectados a una rama
    • git remote -v
  • INICIAMOS GIT
    • git init
  • VAMOS a GITHUB (arriba) y creamos un nuevo Repo de nombre SISI
    • crear nuevo repositorio
    • agregar nombre SISI
    • tocar el botón crear
  • Conexión remote add
    • Copiamos el comando: git remote add origin https://github.com/HX-MLuquez/SISI.git
    • lo pegamos en nuestra consola y damos ENTER
  • Renombre a rama MASTER -> MAIN
    • Copiamos el comando: git branch -M main
    • lo pegamos en nuestra consola y damos ENTER
  • ADD - ARMAR el commit (la caja a salvar). Nuestra versión (nuestro back)
    • git add .
  • COMMIT - Poner nombre a la caja y git lo identifica con un hash
    • git commit -m "iniciando nuestro primer subida de un repo a GITHUB"
  • PUSH - subir, empujar lo guardado hacia arriba, pero este push por primera y única vez es un push especial
    • git push -u origin main
    • para futuros push solo se debe hacer: git push
  • Ver status
    • git status

    A partir de ahora para cada vez que deseamos subir cambios haremos los 3 pasos

  • ADD - ARMAR el commit (la caja a salvar). Nuestra versión (nuestro back)
    • git add .
  • COMMIT - Poner nombre a la caja y git lo identifica con un hash
    • git commit -m "iniciando nuestro primer subida de un repo a GITHUB"
  • PUSH - subir, empujar lo guardado hacia arriba
    • git push
